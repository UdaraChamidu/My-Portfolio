## Anthropic's Claude Code gains 'auto mode' for permissions.

Anthropic's Claude Code now features an "auto mode" that allows it to select its own permissions, offering a faster yet safer alternative to manually approving prompts. This new mode aims to streamline code generation by intelligently managing permissions, addressing a user tendency to bypass manual approvals.
